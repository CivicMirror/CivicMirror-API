# Kansas Election Results — Research Notes

**Site:** https://sos.ks.gov/elections/election-results.html
**Statistics:** https://sos.ks.gov/elections/election-statistics.html
**Operated by:** Kansas Secretary of State
**Researched:** March 4, 2026
**Status:** Public, no authentication required

---

## Overview

Kansas provides election results through the Secretary of State's website in PDF and Microsoft Excel formats. Results are available from 2004 onward online, with older data available upon request.

---

## Data Access

### Election Results
- **URL:** https://sos.ks.gov/elections/election-results.html
- Click year to view available election and precinct-level results
- PDF and Excel formats available
- Coverage: 2004–present online

### Election Statistics
- **URL:** https://sos.ks.gov/elections/election-statistics.html
- Interactive charts and visualizations
- Voter registration by party
- Provisional ballot statistics
- Advance voting (in-person and mail) statistics
- Data tracked by party affiliation since 2020

### Historical Data
- Pre-2004 data available upon request
- Contact: 800-262-VOTE (8683) or election@ks.gov

---

## API Access

No public REST API identified. Data access is through:
1. PDF and Excel downloads from election results page
2. Interactive statistics visualizations
3. Request for historical/precinct data

---

## Notes

- 105 counties in Kansas
- Results in PDF format include official vote totals with candidate names, votes, and percentages
- Advance mail ballot tracking by party affiliation available since 2020
- Kansas allows provisional ballots for any reason (higher percentage than national average)
